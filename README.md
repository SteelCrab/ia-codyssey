# IA Codyssey 🚀

Python 개발환경 설정부터 시작하는 코딩 학습 프로젝트

## 📖 프로젝트 소개

IA Codyssey는 Python 개발환경 설정과 기초 프로그래밍을 단계별로 학습할 수 있는 교육용 프로젝트입니다. 
실무에서 필요한 개발환경 구축부터 웹 프레임워크 이해까지 체계적으로 학습할 수 있습니다.

## 📚 과정 구성

### 세계적인 브랜드 반달곰 커피
- **반달곰 커피의 시작**: Python 설치 및 환경 구성
  - Python 설치 및 환경변수 설정
  - Flask 패키지 설치
  - Hello 함수 구현
  - VS Code 및 확장 프로그램 설치
  - Python 웹 프레임워크 비교 분석

## 🛠 기술 스택

- **언어**: Python 3.8+
- **패키지 관리**: pip, venv
- **웹 프레임워크**: Flask, Django, FastAPI
- **에디터**: Visual Studio Code
- **운영체제**: macOS, Linux
